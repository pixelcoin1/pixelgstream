# PixelG Stream

A scalable Web3 livestream platform with React + Livepeer + Supabase.

## Features
- Live streaming via Livepeer
- Wallet login & VIP check
- NFT badge minting (ERC721)
- Multilingual UI

## Tech Stack
- React + TailwindCSS
- ethers.js
- Supabase
- Livepeer

---

## Run locally
```bash
npm install
npm run dev
```
